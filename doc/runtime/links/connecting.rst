Managing a Link's Graph of Connections
=============================================

The `LinkGraphManager` class encapsulates all the logic for deciding how to
connect the sources of a `LinkNode's <LinkNode>` subgraph to its targets, and
how to maintain those connections as source and target nodes split and merge.

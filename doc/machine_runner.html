Machine Runner¶

class dist_zero.machine_runner.MachineRunner(machine_config)[source]¶

For running A NodeManager on a machine inside a runloop. Real time is passed in, and messages are read from os sockets.

configure_logging()[source]¶

Configure logging for a MachineController

node_manager = None¶

The NodeManager underlying this MachineRunner

runloop()[source]¶

Enter a runloop for the contained NodeManager.

In each iteration of the loop, pass the real elapsed time to NodeManager.elapse_nodes on the NodeManager and pass messages from sockets to NodeManager.handle_message and NodeManager.handle_api_message.

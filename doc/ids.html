Unique Ids¶

Node and MachineController instances (among other things) often need to be referred to by id. This module centralized all the functions for creating and working with unique ids.

dist_zero.ids.new_id(prefix)¶

Generate unique ids for things.

Parameters:prefix (str) – A prefix to prepend to the new id.
Returns:A unique id.
Return type:str

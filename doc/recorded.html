Using Recordings and User-input Generators to Simulate User Input¶

class dist_zero.recorded.RecordedUser(name, time_action_pairs=None)[source]¶

In tests, it can be helpful to generate recordings of user interactions and play them back against nodes in order to generate input for a distributed system. Instances of this class represent the recorded user.

elapse_and_get_messages(ms)[source]¶
Parameters:ms (int) – The number of milliseconds to elapse.
Yield:pairs (ms, msg) where ms is the amount of elapsed time, and msg is a Message generated at that time.

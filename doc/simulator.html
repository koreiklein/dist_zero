Simulated Spawner¶

A simulated spawner will run a simulation of a distributed system entirely within the parent python process.

class dist_zero.spawners.simulator.SimulatedSpawner(system_id, random_seed='random_seed')[source]¶

A class for creating instances of MachineRunner each in its own container to simulate distinct machines in tests.

Tests should typically create a single instance of this factory, and use it to generate all their MachineController instances. Then, this factory class will simulate the behavior of a real-life network during the test.

Parameters:
  • system_id (str) – The id of the overall simulated distributed system.
  • random_seed (str) – A random seed for all randomness employed by this class.

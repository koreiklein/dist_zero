Virtual Spawner¶

A virtual spawner will run a distributed system as a collection of virtual machines all residing on the same physical host.

DistZero’s current virtualization technology is docker; virtualized distributed systems should be run with the docker spawner.

class dist_zero.spawners.docker.DockerSpawner(system_id)[source]¶

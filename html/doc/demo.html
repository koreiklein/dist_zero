demo_single_node_hits_sender_limit()[source]¶

Spawn a single sum node, and keep adding input nodes that send to it. When running this demo, a developer should – through a log analysis tool – be able to chart the number of senders on that node over time, watching as it continues to increase, until eventually it hits the node’s limit for number of senders and some indication of the limit appears on the chart.
